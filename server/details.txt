HOTEL MANAGEMENT APP - RUN GUIDE

1. Requirements:
Install Node.js.
Check with:
node -v
npm -v

2. Start backend:
Open terminal in the server folder:

cd server
npm install
node server.js

Expected message:
Server running on http://localhost:3000

Test in browser:
http://localhost:3000/

3. Start frontend:
Open another terminal in the client folder:

cd client
npx http-server
OR
http-server

Open the URL shown in the terminal, usually:
http://localhost:8080/
or sometimes:
http://localhost:8081/

4. Important:
Backend runs on:
http://localhost:3000

Frontend runs on:
http://localhost:8080 or http://localhost:8081

Both must be running at the same time.

5. Current API routes:
POST /signup
POST /login
GET /users
PUT /users/:id
PUT /users/:id/balance
DELETE /users/:id
POST /clients/connect
GET /clients

Example backend URL used in app.js:
const API_URL = "http://localhost:3000";

6. Data is stored in:
server/database.json

7. Main functions:
- User sign up and login.
- User logout and login status check.
- View all users.
- Update username and password.
- Update user balance.
- Delete account with role-based permission.
- Register opened client pages.
- View connected client count.
- Store all data in database.json.
